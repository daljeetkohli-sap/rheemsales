# Rheem Sales Call Steps Planner

Static CRM-style prototype built from the requirements workbook. It includes:

- Must Have, Should Have, and Other Haves requirement boards
- Dummy customer, stock, sales, promotion, training, competitor, quotation, and reporting data
- Step-of-the-call workflow for visit planning and execution
- Activity calendar for cycle planning

The app runs directly from `index.html` and is deployed through GitHub Pages.
